1. XPath - Traversing(Three types): Forward(/ or // like //div/input[1]), Backward(/..)
            -> Methods to locate the dynamic element:
            -> 1. Identify static element. 2. Move to common parent. 3. From the common parent fetch the dynamic element

2. XPath - Sibling: Following Sibling and Preceding Sibling
            -> Example:
                <tr>
                    <td> U/A </td>  // Preceding sibling
                    <td> Dhurandar 2 </td>  // Static
                    <td> 4 star </td> // following sibling
                    <td> 10 cr </td> // following sibling
                </tr>
                -> to know the second following sibling and preceding sibling
                -> //td[text()='Dhurandar 2']//following-sibling::td[2]
                -> //td[text()='Dhurander 2']//preceding-sibling::td[1]
